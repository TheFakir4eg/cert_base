работа модального окна "подтверждение удаления"
    Пользователь нажимает на кнопку с data-confirm="true".
    JavaScript перехватывает событие click.
    Он читает data-title, data-confirm-text и data-confirm-form-id.
    Заполняет универсальное модальное окно этими данными.
    При нажатии на кнопку "Да" в модальном окне, JavaScript получает ссылку на форму (по id) и вызывает form.submit(), что отправляет POST-запрос на сервер.
    Маршрут places_routes.py обрабатывает запрос

 как попасть в бд внутри docker 
    Сначала узнаём имя контейнера БД
        docker ps
    Подключаемся к MySQL(в нашем случае mariadb) внутри контейнера
        docker exec -it cert_base_db mariadb -u fakir -p
    Смотрим базы
        SHOW DATABASES;
    Выбираем нужную:
        USE cert_db;
    Смотрим пользователей
         SELECT id, login  FROM users;
